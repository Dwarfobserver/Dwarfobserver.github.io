Tentacle Slicers

One year ago, I created Tentacle Slicers in pair, a 2D top-down game playable alone or with a friend (on the same keyboard) where you must survive against multiples ennemies. Powerups boost you character and give him new abilities : powerful projectiles, attacking while moving, invincibiliy, disable ennemies, ...

Tentacle Slicers has more than 8000 lines of code in C#. He is not based on any game engine. I created the engine architecture and his systems : collisions (simple forms and lines of sight), animations and artificial intelligence with pathfinding. I drawed all graphic assets of the game (images, maps, effects and animations). I also developped I file parser to create levels.

Here is the game's source code. GitHub hasn't been used for his developpement and is only used to store the final project. A Youtuber has accepted to test our game :)

Car Crash Boom !!! Yeah !

I a school course about video game conception, I participated to the developpement of 'Car Crash Boom ! Yeah !!!', a local multiplayer racing game developped on Unity where duos are fighting each other to finish first. We have three fighting races and two cooperation races for one final map.

In particular, I created a generic input system and a playable fighting race (on a beach). I also look how to do the lightning and the link between differents races for a single map.
