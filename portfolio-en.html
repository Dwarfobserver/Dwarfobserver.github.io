My portfolio

Tentacle Slicers

One year ago, I created Tentacle Slicers in pair, a 2D top-down game playable alone or with a friend (on the same keyboard) where you must survive against multiples ennemies. Powerups boost you character and give him new abilities : powerful projectiles, attacking while moving, invincibiliy, disable ennemies, ... One can save his progress in-game, which will keep the game's state at the last level beginning.

Tentacle Slicers has more than 8000 lines of code in C#. He is not based on any game engine. I created the engine architecture and his systems : collisions (simple forms and lines of sight, world partitionning), animations, artificial intelligence with pathfinding and the file system for creating new levels and loading games. I drawed all graphic assets of the game (images, maps, effects and animations).

I put a lot of my time in this project and I learned a lot of things. I'm pretty proud of it (even if it has still few bugs) and it made me (more) enthusiast about code organisation and game engines.

Here is the game's source code. GitHub hasn't been used for his developpement and is only used to store the final project. A Youtuber has accepted to test our game :)

Car Crash Boom !!! Yeah !

In a school course about video game conception, we were four to make 'Car Crash Boom ! Yeah !!!' alive, a local multiplayer racing game developped on Unity where duos are fighting each other to finish first. We have three fighting races and two cooperation races for one final map.

In particular, I created a generic input system and a playable fighting race (on a beach). I also look how to do the lightning and the link between differents races for a single map.

T-Arena

I participate at a Game Jam where our team of six persons developped 'T-Arena', a 2D top-down game for two persons playing in turns : two teams of hero must fight on a tiled map.

I created the game sounds, the main loop logic and the input handling (for keyboards and gamepads), with in particular an algorithm to select cases where we want to move or attack.
